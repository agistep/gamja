


# README 



## Configuration

Gamja 는 기본적인 설정을 classpath:/gamja.yml 을 제공한다. 

## Configuration 우선순위

1. core!/gamja-default.yml
2. 사용자 정의 classpath:/gamja.yml

## How to Override Configuration

Custom Config gamja.yml 를 정의한다. 


## gamja.yml FilePath 변경
```shell

java sdfdsf -Dgamja.config.path=

```
